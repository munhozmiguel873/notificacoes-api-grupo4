# Estrutura do Processo Scrum

Este documento detalha a aplicação dos conceitos ágeis em nosso projeto atual.

## 📅 Definição de Ciclo
|         Conceito         |              No nosso projeto            |
| ------------------------ | ---------------------------------------- |
| **Sprint**               |           Período de 2 semanas           |
| ------------------------ | ---------------------------------------- |
| **Sprint Planning**      |    Início da sprint: escolher o que      |
|                          |                 fazer.                   |
| ------------------------ | ---------------------------------------- |
| **Sprint Review**        |   Final da sprint: mostrar o que foi     |     
|                          |                 feito.                   |                                                                
| ------------------------ | ---------------------------------------- |
| **Sprint Retrospective** |      O que funcionou, o que melhorar     |
| ------------------------ | ---------------------------------------- |
| **Daily Standup**        |  Breve check-in no início de cada aula   |
