# Definition of Done (DoD)

Uma funcionalidade só é considerada PRONTA quando:

1. [✅] O código está funcionando (testado manualmente)
2. [✅] Segue o padrão MVC + Services
3. [✅] Erros são tratados com classes customizadas
4. [✅] Validações estão implementadas
5. [✅] A rota está documentada no Swagger
6. [✅] Pelo menos um outro membro revisou o código
7. [✅] Commit feito com mensagem descritiva
8. [✅] Card movido para "Concluído" no Kanban
